# Publication Policy

このリポジトリは、就活用の公開ポートフォリオとして運用する前提で構成しています。

## 基本方針

- 元の授業課題や業務データは、そのまま公開しません
- 公開利用可能な別データに差し替えて再構成します
- 企業名、サービス名、内部指標名、個票識別子は持ち込みません
- notebook の Colab 実行痕跡、secret、Drive パス、ローカルパスは削除します

## 公開しないもの

- 講座で配布された課題文の本文
- スライドや教材の図表・文章
- 業務データ、顧客データ、社内向け CSV / Excel
- 受講者限定で提供された notebook やデータ一式
- token、API key、secret、個票識別子

## 公開するもの

- 自分で整理し直した README
- 公開データを前提にした notebook
- 再現手順を支える最小コード
- 背景、改善方針、考察

## 各プロジェクトで守ること

1. データ出典を README に明記する
2. 利用条件が曖昧な素材は採用しない
3. notebook を開かなくても README で価値が伝わるようにする
4. 数値や図は公開データまたは公開可能な再現結果に限定する

## このポリシーの意図

インターン選考で見せたいのは、配布物の再掲ではなく、問題設定を理解して自分の形に再構成できる力です。  
そのため、このリポジトリでは「安全に公開できること」と「自分の工夫が説明できること」を優先しています。
